设计模式-简单工厂

发表于2019-06-01|设计模式
|阅读量:

简单工厂模式严格来说并不是一个设计模式，反而较像是一种编程习惯。

定义

简单工厂模式(Simple Factory Pattern)：又称为静态工厂方法(Static Factory Method)模式，它属于类创建型模式。在简单工厂模式中，可以根据参数的不同返回不同类的实例。简单工厂模式专门定义一个类(工厂类)来负责创建其他类的实例，被创建的实例通常都具有共同的父类。

结构

通用写法

//客户端
public class Client {

    public static void main(String[] args) {
        IProduct product = SimpleFactory.create("A");
        if (product!=null) {
            product.desc();
        }
    }

}

//抽象产品
interface IProduct {
    void desc();
}

//具体产品A
class ProductA implements IProduct {

    @Override
    public void desc() {
        System.out.println("我是产品A");
    }
}

//具体产品B
class ProductB implements IProduct {

    @Override
    public void desc() {
        System.out.println("我是产品B");
    }
}

//工厂类
class SimpleFactory {
    public IProduct create(String name) {
        if ("A".equals(name)) {
            return new ProductA();
        } else if ("B".equals(name)) {
            return new ProductB();
        } else {
            return null;
        }
    }
}

工厂类的create方法根据传入参数的不同创建不同的产品。

优化

1.工厂类的create方法改为静态方法

为方便调用，将工厂类的create方法改为静态方法。

class SimpleFactory {
    public static IProduct create(String name) {
      //
    }
}

public static void main(String[] args) {
    IProduct product = SimpleFactory.create("A");
}
2.使用反射-1

当具体的产品类继续增加时候，需要修改工厂类的create方法逻辑，且逻辑会越来越复杂，不符合开闭原则，可使用反射对其进行优化。

class SimpleFactory {
    public static IProduct create(String className) {
        if (isNotBlank(className)) {
            try {
                return (IProduct) Class.forName(className).newInstance();
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        return null;
    }
}

//客户端
public static void main(String[] args) {
    IProduct product = SimpleFactory.create("com.my.simplefactory.ProductA");
    if (product!=null) {
        product.desc();
    }
}
3.使用反射-2

上述优化后代码需要传入字符串，而且还需要强制转型，可继续进行优化。

class SimpleFactory {
    public static IProduct create(Class<? extends IProduct> clazz) {
        if (clazz != null) {
            try {
                return clazz.newInstance();
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        return null;
    }
}

//客户端
public static void main(String[] args) {
    IProduct product = SimpleFactory.create(ProductA.class);
    if (product!=null) {
        product.desc();
    }
}
